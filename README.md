# FleetTime (temporary)

A web-based vehicle scheduling system for teams or families.

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Supabase (PostgreSQL + Auth)
- Vercel (hosting)

## Setup
```bash
npm install
npm run dev
npm run build
```

## Environment Variables
```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
APP_ENV=
```
